# FollowTX

## Version for Monero

### Install 

#### Required

- Nodejs
- npm
- Nginx
- Redis
- Mongodb
- RoboT3 (local)
- Monero (daemon, wallet-rpc)
  
        git clone https://github.com/FollowTX/FollowTX-Monero.git
        cd FollowTX-Monero
        npm install
        cp .env.exampple .env
        nano .env # or vim .env
        npm run check
        npm run keys
